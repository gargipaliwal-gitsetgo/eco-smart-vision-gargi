# EcoSmart Vision ♻️

**EcoSmart Vision** is a modern, visually stunning Smart Waste Management conceptual web application. It is designed to demonstrate an AI-powered waste segregation workflow through a premium, interactive user interface.

## 🌟 Features

- **Premium UI/UX:** Built with modern web design principles including soft **Glassmorphism**, floating geometric background animations, and custom typography (Outfit).
- **Interactive Segregation:** Features a Drag & Drop interface where users can upload an image of their waste item to "classify" it.
- **Dynamic 3D Dashboard:** A scroll-triggered analytics dashboard featuring 3D flip-cards with custom `cubic-bezier` animations.
- **Real-Time Bin Status:** Tracks scans during the session and simulates the physical bin's capacity, updating live metrics for Plastic, Wet Waste, and Metal categories.

## 💻 How to Run Locally

Since this is a client-side only web application, absolutely no build tools or servers are required!

1. Clone or download this repository.
2. Navigate to the project folder.
3. Double click on `index.html` to open it in your default web browser.
